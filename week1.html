Vika 1

Markmið vikunar:

  • Læra grunninn í html
  • Búa til algeran grunn af vefsíðu
  • Koma grunn vefsíðunni inná Github

Framkvæmd vikunar:

  • Til að byrja með fylgdi ég þessu myndbandi til að búa til github vefsíðu: How to host a website on Github
  • Lenti í veseni með gitbash, eftir að ég bjó til repositry þá gat ég ekki pushað á master branchið, endaði á að uploada bara index.html file-inum beint á github með upload files takkanum.
  • Þegar grunn skráin var komin inn á Github þurfti að fara í settings á repository og finna "Github Pages" og þar var hægt að nálgast link að vefsíðunni.
  • Ég eyddi miklum tíma í að kóða í Github editor sem voru mistök þar sem breytingar voru 1-2 mínútur að koma inn á vefsíðuna og ég var ennþá bara að læra grunninn svo það voru gerðar mjög litlar breytingar í einu. Ég var fljótur að færa mig yfir í Atom umhverfið: Atom forritið Hægt er að tengja Atom umhverfið við Github aðgang og þannig hægt að gera margar breytingar locally og svo pusha auðveldlega á Github.
  • Spent a lot of time learning the basics of html, I found a great website for absolute beginners that I read through: HTML for beginners
